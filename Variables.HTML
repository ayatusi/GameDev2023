javascript "var" example

x, y, and z are variables

var x = 12;
var y = 8;
var z = x + y;

"var" can be replaced with either "let" or blank

javascript "const" example

price1, price2, and total are variables.

const price1 = 5;
const price2 = 6;
let total = price1 + price2;

creating a variable, assigning a value to it then displaying it

example code:

let carName = "Ferrari";
document.getElementById("car").innerHTML = carName;
